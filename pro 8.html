Major Browsers' Debugging Tools Normally, you activate debugging in your browser with F12, and select "Console" in the debugger menu. Otherwise follow these steps: Chrome Open the browser. From the menu, select "More tools". From tools, choose "Developer tools". Finally, select Console. Firefox Open the browser. From the menu, select "Web Developer". Finally, select "Web Console". Edge Open the browser. From the menu, select "Developer Tools". Finally, select "Console". Opera Open the browser. From the menu, select "Developer". From "Developer", select "Developer tools". Finally, select "Console". Safari Go to Safari, Preferences, Advanced in the main menu. Check "Enable Show Develop menu in menu bar". When the new option "Develop" appears in the menu: Choose "Show Error Console". Did You Know? Debugging is the process of testing, finding, and reducing bugs (errors) in computer programs.
